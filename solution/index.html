Welcome to KANBAN - Your task manager

KANBAN has a lot of interesting features like:
- Add tasks to 3 different lists sorted by task status
- Move a task from one list to another by pressing the alt key with the desired list number
- Move a task from one list to another by dragging
- Editing a task by double-clicking on the task
and more
